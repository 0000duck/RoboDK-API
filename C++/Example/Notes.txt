This sample project implements the RoboDK API for C++ 
and it has been developed using Qt libraries and Qt Creator.

The RoboDK API involves the files:
    robodk_api.h
    robodk_api.cpp

To Download Qt:
https://www.qt.io/download

This project does not require any specific DLL dependencies.

It has been tested with:
Qt 5.7
Qt 5.10

More information about the RoboDK API here:
https://robodk.com/doc/en/RoboDK-API.html
https://robodk.com/doc/en/PythonAPI/index.html

More information about RoboDK post processors here:
https://robodk.com/help#PostProcessor
